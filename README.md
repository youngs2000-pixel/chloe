# 🐱 아기고양이 임시 집사 (Baby Cat Caretaker)

7일 동안 아기고양이를 돌보는 귀여운 브라우저 육성 게임이에요.
아이와 함께 만들었어요!

👉 **[지금 플레이하기](https://youngs2000-pixel.github.io/chloe/)** (아래 GitHub Pages 설정 후 활성화돼요)

## 🎮 게임 방법

- 고양이의 **이름**과 **성격**을 골라요. (활발이 · 얌전이 · 겁쟁이 · 먹보 · 애교쟁이)
- 매일 **밥주기 / 아이템 사용 / 쓰다듬기** 중 하루 3번 행동할 수 있어요.
- **배고픔 · 기분 · 건강 · 졸림 · 친밀도**를 잘 관리해요.
- 매일 밤 주인이 찾아와 **별점(⭐)** 을 줘요. 별점으로 상점에서 아이템을 사요.
- 7일 뒤 결과에 따라 **3가지 엔딩** 이 나와요!

## 🐈 특징

- 고양이 표정이 상태에 따라 바뀌어요 (ASCII + 이모지).
- 음식·아이템마다 다양한 반응 대사가 나와요.
- 까칠한 날, 낯선 소리, 창가 햇살 등 랜덤 이벤트 10종.
- 설치 없이 브라우저에서 바로 플레이 — 단일 `index.html` 파일.

## 💻 직접 실행하기

`index.html` 파일을 더블클릭해서 브라우저로 열기만 하면 돼요.

## 📁 파일

| 파일 | 설명 |
|------|------|
| `index.html` | 브라우저용 게임 (HTML + CSS + JavaScript) |
| `아기고양이_임시_집사.py` | 원본 파이썬(터미널) 버전 |
| `아기고양이 임시 집사 게임 개요.docx` | 기획 문서 |
